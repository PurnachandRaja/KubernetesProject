## Description
This is simple kubernetes project in which I deployed mongodb & mongo-express using K8s manifest files.

## Concepts learned 
1. Deployment, Replicaset, Pod
2. Service (External = loadBalancer Internal = ClusterIP) 
3. Secret
4. ConfigMap
